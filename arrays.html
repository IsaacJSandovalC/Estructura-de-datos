Arrays

¿Qué es un array en programación? Un array es un tipo de dato estructurado que permite almacenar un conjunto de datos homogéneo y ordenado, es decir, todos ellos del mismo tipo y relacionados. Su condición de homogéneo, indica que sus elementos están compuestos por el mismo tipo de dato, y su condición de ordenado hace que se pueda identificar del primer al último elemento que lo compone.

Definiendo un array en C++:

Primero empezaremos declarando nuestra variable como tipo str:

str array;

seguido a eso, el vamos a indicar de cuanto sera nuestra cantidad maxima de letras que tendra el array.

str array [10];

Tambien podemos colocarle la cantidad de texto que queremos ya de manera pre determinada.

array [10]; array[0] = 1; array[1] = 3; array[2] = 10;

Asi como se muestra en el ejemplo anterior, asignamos al indice cero '0' el numero uno '1', al indice uno '1' el valor tres '3' y al indice dos '2' el valor diez '10'
